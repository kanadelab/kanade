「奏でる日常の音色」

【メインキャラクター】
かなで
（「奏」が正しいですが他のゴーストとかぶっちゃたので。）

【ゴーストの説明】
立ち寄ったカフェは、まだ開店準備中でした。
お茶とお菓子が大好きな女の子と準備中のカフェで過ごすソロゴースト。
トークはお茶とお菓子に関係したものが主です。
便利な機能もそろえており、お役立ちなゴーストです。

【推奨動作環境】
Microsoft Visual C++ 2008 ランタイム
最新版のSSP

【操作説明】
カーソルの変わらないところでダブルクリックするとメニューを開きます。
スカートをダブルクリックするとトークします。
腰のリボンをクリックすると設定した機能を素早く開けます。
その他はなでたり、つついたり。

ショートカットキー:
T:ランダムトーク
M:メニュー
O:各種設定
C:コミュニケート

【ゴーストをより楽しんで頂くために】
・名前を教えよう
はじめて起動した時には、呼び名を聞かれます。
教えたとおりに呼んでくれるので、好きな呼び名を教えてください。

・更新をインストールしよう
インストールしたばかりのゴーストは、少し内容が古いかもしれません。
キャラクターを右クリックして出るメニューの「オンラインアップデート」で
最新の情報をインストールしましょう。

・設定でもっと使いやすく
メインメニューから「各種設定」を選んでこのゴーストに関する設定ができます。
呼び名を教えたり、おしゃべりの間隔を変えたり…
メインメニューに表示する機能を好きなように変えることもできます。
更新を自動でチェックして、メインメニューで教えてくれる機能もあります。

・意見やネタを送信しよう
メインメニューの「作者にメッセージを送信する」をクリックして
作者にメッセージを送ることができます。
ご意見・ご感想・その他雑談やリクエストなどお気軽にどうぞ。お返事は作者のブログで。
また、設定で「未対応コミュニケートを送信」を有効にして
話しかけても対応していなかったメッセージをリクエストとして自動的に送信することができます。

【セーブデータを取り出す際の注意】
里々のセーブデータのsatori_savedata.txtの他に、
「奏でる日常の音色」が独自に作成したセーブデータkanade_core.txtも取り出す必要があります。

【機能の紹介】
「天気予報」指定した地域の天気情報を取得します。
「ランチャ」ファイルやURLを登録すればゴーストから開けます。
「インターネット検索」検索ワードを入力して結果をブラウザで開きます。
「ニュース」ニュースを取得しゴーストから観覧できます。
「音楽再生」簡単なプレイリストを作ったり音楽ファイルを再生できます。
「メッセージ送信」作者にメッセージを送信します。
「更新確認」有効にすると起動時に自動で更新の有無を確認します。
